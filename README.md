# -Monitoreo-de-Sensores-Ambientales-con-MQTT
Monitoreo Ambiental IoT en Tiempo Real (RTEM). Proyecto que implementa una solución completa de captura y visualización de datos (Temperatura, Humedad, UV) utilizando MQTT para la ingesta en tiempo real, InfluxDB como base de datos Time Series y Grafana para la monitorización interactiva.
